59.螺旋矩阵

题目

给你一个正整数 n ，生成一个包含 1 到 n2 所有元素，且元素按顺时针顺序螺旋排列的 n x n 正方形矩阵 matrix 。

示例 1：

输入：n = 3

输出：[[1,2,3],[8,9,4],[7,6,5]]

示例 2：

输入：n = 1

输出：[[1]]

提示：

  • 1 <= n <= 20

思路

  本题要求将n*n个数从左上角开始按顺时针的顺序依次存入矩阵中。因为矩阵是方的且按照时针顺序存入数据，所以每走完整的一圈可以视为完成一次循环，每个矩阵只需完成n/2次循环。特别地，如果矩阵是奇数次矩阵还要单独考虑中心的位置。

  我们将每次循环分为上右下左四个步骤，这四个步骤划分规则是左闭右开，也就是说包含左边的位置不包含右边的位置。每次循环结束刚好完成一圈，且下次循环开始位置在对角线上的下一个位置。
