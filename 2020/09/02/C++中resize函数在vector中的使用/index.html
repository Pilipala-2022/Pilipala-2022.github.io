C++中resize函数在vector中的使用
发表于2020-09-02|更新于2020-09-02|随笔

  在C++中typeid函数用于查看表达式的类型信息，包含在头文件 typeinfo 中。resize可以根据一个vector容器的大小为另一个容器开辟空间

vector的类型信息
typeid(layer_neuron_num).name() = class std::vector<int,class std::allocator >

vector.size()的类型信息
typeid(layer_neuron_num.size()).name = unsigned __int64

layer_neuron_num的大小是3，表示包含三个int类型的数据
layer_neuron_num.size() = 3

vector的类型信息
typeid(layer).name() = class std::vector<class cv::Mat,class std::allocator >

vector.size() 的类型信息
typeid(layer.size()).name() = unsigned __int64

layer的大小为3，表示有3个Mat类型的数据
layer.size() = 3
